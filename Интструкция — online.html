Создание промежуточного файла



Содержание инструкции:
  1. Создание нового файла Excel
  2. Шаблон таблицы
  3. Описание полей
  4. Синхронизация данных

1. Создание нового файла Excel

 Открываем Excel и создаем новый файл (Пустая книга)



На экране мы должны увидеть пустую табличку, как на картинке снизу




2. Шаблон таблицы

 Теперь нам нужно подготовить наш промежуточный файл, чтобы в будущем корректно вносить все наши данные.
Для этого нужно скачать следующий Excel файл, в нем есть нужные нам поля, а также их описание.



В скачанном файле нас интересует первая строка с названиями колонок.
По нажатию на номер строки или с помощью левой кнопки мышки выделяем всю строку.
Нажимаем на выделенную строку правой кнопкой мышки и выбираем пункт “Копировать”
(также для копирования можно использовать сочетание клавиш CTRL + C)



Теперь открываем наш созданный пустой файл и вставляем в него скопированную строку.




*** При вставке строки важно выбрать самую верхнюю левую ячейку A1 ***

3. Описание полей

В скачанном файле "Шаблон таблицы.xlsx" есть отдельная страница с описанием всех полей таблицы.

Также для вашего удобства, описание полей, обязательных и справочных колонок приведено в таблице ниже.

  • Красным + фон - колонки - обязательные при загрузке файла
  • Красным - колонки - редактируемые, - обязательные при создании товара|услуги
  • Зеленым - колонки - редактируемые
  • Оранжевым - колонки - редактируемые, “логические” - влияют на другие
  • Серым - колонки - “справочные” для информации, вспомогательные


Поле Описание
d_ordExl - порядковый номер строки при выгрузке в файл
d_firmShortName - краткое название фирмы 
d_firmID - идентификатор фирмы 
d_ctuName - название “Категории Маркета” ... _изменить_категорию_ (подробнее в Инструкции)
d_ctuID - идентификатор “Категории Маркета”
